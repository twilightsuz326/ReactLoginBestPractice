# React + Laravel + Sanctum Login Best Practice

## Overview
- Laravel 10 + React + Sanctum でのログインのベストプラクティスを探る。

## cmd
```bash
laravel new ***
# .env 修正
php artisan migrate
npm install
npm install react react-dom react-router-dom
npm install -D @vitejs/plugin-react 
# vite.config.js 修正

npm run dev
```
